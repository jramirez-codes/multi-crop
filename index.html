The Easiest Way to Crop Multiple Regions From One Image

Multi-Crop is a free online image cropping tool that lets you select and export multiple crop regions from a single image. Unlike traditional croppers that only handle one area at a time, Multi-Crop lets you draw as many rectangular selections as you need, name each one, and download them all as individual PNG files with one click.

How It Works

  1. Upload, drag and drop, or paste an image from your clipboard.
  2. Draw rectangles on the image to define crop regions.
  3. Name each region in the sidebar for organized file exports.
  4. Toggle automatic background removal to get clean cutouts.
  5. Click Export Crops to download every region as a separate PNG.

Key Features

  • Crop multiple areas from one image at once — no repeated uploads.
  • Automatic background detection and removal using edge-pixel analysis.
  • Pan and zoom for precise selections on large images.
  • Runs 100% in your browser — your images are never uploaded to a server.
  • No signup, no watermarks, completely free.
